📖 Comment obtenir l'URL d'un fichier Claude ?
  1. Dans le chat Claude, appuie sur la pièce jointe à télécharger
  2. Le fichier s'ouvre — la barre d'adresse montre claude.ai/public/artifacts/…
  3. Appui long sur la barre d'adresse → Copier
  4. Reviens ici et colle l'URL ci-dessous
⚙ Comment utiliser CE téléchargeur correctement ?
  1. Ne pas télécharger ce fichier et l'ouvrir localement (les CORS vont bloquer)
  2. Cliquer sur la pièce jointe « downloader » directement dans Claude
  3. Le téléchargeur s'ouvre à l'URL claude.ai/public/artifacts/… dans ton navigateur
  4. Là, fetch sera same-origin avec les autres fichiers claude.ai → ça marchera
